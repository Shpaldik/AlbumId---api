# 🌟 Shpaldik  

A sleek and responsive **album search** application built with **Vue 3, PrimeVue, and Vite**.  

## 🚀 Features  
✔️ Search albums by ID  
✔️ Dark mode support 🌙  
✔️ Interactive and responsive UI  
✔️ Efficient data handling  

## 🛠️ Tech Stack  
- **Frontend:** Vue 3
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS
